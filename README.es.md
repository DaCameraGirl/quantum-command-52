## Inicio rápido

Desde la raíz del repositorio en PowerShell:

```powershell
python grants.py rank
python housing_violations.py summarize
python shell_catalog.py estimate
python quantum_portfolio.py --capital 1000
python qml_signal_engine.py --capital 1000
python strict_macro_quantum_v10.py --preflight
```

Los resultados se guardan en la carpeta `output`.

## IBM Quantum

El script estricto V10 requiere dependencias empresariales y acceso a IBM Runtime.

```powershell
py -3.11 strict_macro_quantum_v10.py --preflight
```

Guarda los secretos en `.env`. No subas claves privadas de IBM, Alpaca, base de datos u otras.

### Investigación en papel con Alpaca

El trading en vivo está bloqueado intencionalmente. Solo vista previa/envío en papel:

```powershell
py -3.11 strict_macro_quantum_v10.py --bankroll 1000 --preview-alpaca-orders
py -3.11 strict_macro_quantum_v10.py --bankroll 1000 --preview-alpaca-orders --submit-paper-orders
```

## Configuración del panel web

```powershell
cd web-dashboard
npm install
py -3.11 -m pip install -r requirements.txt
py -3.11 server.py
```

```powershell
cd web-dashboard
npm run dev
```

Abre http://127.0.0.1:5173

## Modo demo local

El acceso directo inicia el backend con:

```text
REPO52_DEMO_MODE=true
APP_ENV=development
REQUIRE_ALEMBIC_MIGRATIONS=false
```

Usa BD local ignorada: `web-dashboard/data.db` — elimínala para restablecer filas de demo.

## Base de datos de producción

```text
DATABASE_URL=postgresql://postgres:postgres@127.0.0.1:5432/quantum_command_52
DATABASE_POOL_MIN=1
DATABASE_POOL_MAX=10
JWT_SECRET=replace-with-at-least-32-random-characters
ALLOWED_ORIGINS=http://127.0.0.1:5173,http://localhost:5173
RATE_LIMIT_AUTH_PER_MINUTE=12
RATE_LIMIT_API_PER_MINUTE=120
```

```powershell
cd web-dashboard
py -3.11 -m alembic upgrade head
```

## Úsalo de verdad

1. Abre los archivos CSV en `data`.
2. Añade oportunidades, recursos, evidencia o artículos reales.
3. Mantén las URL oficiales en las columnas `source_url`.
4. Ejecuta el script correspondiente.
5. Usa el Markdown/CSV generado en `output`.

## Límites

- Ningún script garantiza una beca, beneficio, acuerdo o precio de venta.
- Ningún script es asesoramiento legal o financiero.
- No subas documentos privados de desconocidos ni de sitios web aleatorios.
- Los resúmenes de vivienda/legal son organizadores — consulta asistencia legal calificada para actuar.
